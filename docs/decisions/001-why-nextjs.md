# Why Next.js?

## Problem

Querva requires:

* Modern UI
* Authentication
* Server-side capabilities
* Scalability

## Options Considered

* React + Vite
* Next.js

## Decision

Next.js

## Reason

* Industry standard
* App Router
* Server Components
* Excellent TypeScript support
* Great deployment options
* Suitable for SaaS applications

## Tradeoffs

* Slightly steeper learning curve
* More concepts than plain React
